快速開始

三步完成第一次整理

  1. 新增任務點一下新增按鈕，寫下要完成的事情。
  2. 判斷優先順序選擇是否重要、是否緊急，任務會進入對應象限。
  3. 持續調整拖曳任務即可改變象限；完成後也能從已完成頁面恢復。

原生整合

與 Apple 提醒事項連動

Todo Matrix 使用 Apple EventKit 讀取及更新你授權的提醒事項。建立、完成或恢復任務時，內容與狀態會同步反映在 Apple 提醒事項中。

首次使用時，請依系統提示允許提醒事項存取。你可以隨時在系統設定中調整權限。

常見問題

任務為什麼沒有出現在預期的象限？

象限由「重要」及「緊急」兩個狀態決定。編輯任務，確認這兩個選項，或直接將任務拖曳到想要的象限。

如何恢復已完成的任務？

前往已完成頁面，找到該任務並選擇恢復；任務會回到原本的象限。

為什麼看不到提醒事項？

請到系統設定確認 Todo Matrix 已取得提醒事項權限。若仍無法顯示，可關閉並重新開啟 App。

Todo Matrix 是完整的 GTD 工具嗎？

Todo Matrix 參考 GTD 與時間管理概念，核心是用艾森豪矩陣整理優先順序，並不提供完整的 GTD 工作流程。
