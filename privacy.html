隱私權政策

生效日期：2026 年 7 月 15 日

我們尊重你的隱私

Todo Matrix 不蒐集個人資料，也不出售或分享個人資料。App 不建立使用者帳號、不加入廣告、不使用第三方分析服務，並且不使用追蹤服務。

  • 不需要帳號
  • 沒有廣告
  • 沒有分析
  • 沒有跨 App 或網站追蹤

提醒事項如何儲存

Todo Matrix 透過 Apple EventKit 存取資料，待辦事項存放於你自己的 Apple 提醒事項中，而不是開發者的伺服器。App 只會在你明確授權後存取提醒事項。

iCloud 與 Apple 的服務

若你已為 Apple 提醒事項開啟 iCloud，同步功能由 Apple 提供，相關資料依 Apple 的隱私權政策處理。Todo Matrix 不控制 Apple 的服務或資料處理方式。

權限由你掌握

你可以隨時取消權限。在 iPhone 或 iPad 上，前往「設定」→「隱私權與安全性」→「提醒事項」；在 Mac 上，前往「系統設定」→「隱私權與安全性」→「提醒事項」，然後關閉 Todo Matrix 的存取權。

透過 GitHub Issues 取得支援

支援請求由公開的 GitHub Issues 提供。你提交的內容會由 GitHub 依其政策處理，並可能對外公開。請勿張貼提醒內容、姓名、聯絡方式或其他個人資料。

如果 Todo Matrix 未來加入帳號、廣告、分析、追蹤、開發者伺服器或其他會影響資料處理的功能，本政策會同步更新，並在此頁標示新的生效日期。
